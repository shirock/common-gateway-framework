這是使用 CommonGateway Framework 的初步教學範例。

* 如何建立第一個控制項(controller)。
* 如何建立控制項各行為的視圖(view)。
* 如何在視圖取得控制項傳來的資料(model)。
* 使用 REST client 工具，HTTP request header 設置一行「Accept: application/json」，就會回傳 JSON 資料。
